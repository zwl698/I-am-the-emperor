# 我是皇帝

一个 Go 后端驱动的 Web 皇帝模拟器 MVP。玩家从皇子出生开始，通过剧情选择积累学识、武略、名望、势力等属性；登基后进入回合制治国阶段，在内政、军事、外交和宫廷之间权衡，最终走向盛世、驾崩或王朝崩塌。

## 运行

```bash
./scripts/dev.sh
```

打开 `http://localhost:8080`。

如果需要换端口：

```bash
PORT=3000 ./scripts/dev.sh
```

## 测试

```bash
GOCACHE="$PWD/.gocache" go test ./...
node --check web/app.js
```

## 当前玩法

- 皇子阶段：出生、书院、冬狩、赈灾、夺嫡五段剧情。
- 登基转换：成长属性会折算成国库、粮草、民心、军力、邦交、朝稳、边患等帝国属性。
- 皇帝阶段：每回合选择减赋赈济、推行新法、拨银练兵、遣使联姻或大宴群臣。
- 世界压力：国库、粮草和边患会随回合自然变化，错误决策会触发崩盘结局。
- Web 界面：卡通宫廷风格、属性条、选项卡片、起居注和结局重开。
