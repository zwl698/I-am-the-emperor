# 我是皇帝

一个 Go 后端驱动的 Web 皇帝模拟器。玩家先选择朝代，再从皇子出生开始，通过剧情选择积累学识、武略、名望、势力等属性；登基后进入回合制治国阶段，在六部政务、财政、军务、外交、新法、暗线、派系、省份与危机链之间权衡，最终走向盛世、驾崩或王朝崩塌。

## 运行

```bash
./start.sh
```

脚本会启动后端服务并自动打开前端页面：`http://localhost:8080`。

如果需要换端口：

```bash
PORT=3000 ./start.sh
```

只启动服务、不自动打开浏览器：

```bash
./scripts/dev.sh
```

## 测试

```bash
GOCACHE="$PWD/.gocache" go test ./...
node --check web/app.js
```

## 当前玩法

- 朝代选择：大胤开国、景曜盛世、承平暮年、玄朔边患，每个朝代有不同历史背景、初始国势、派系压力和危机。
- 皇子阶段：出生、书院、冬狩、赈灾、夺嫡五段剧情。
- 登基转换：成长属性会折算成国库、粮草、民心、军力、邦交、朝稳、边患等帝国属性。
- 皇帝阶段：每回合可走民政、财政、军务、外交、新法、暗线、宫廷七类策略。
- 御前操作台：登基后主舞台会出现兵棋沙盘、三司会审、六部调度三个小游戏式面板，可直接点击棋格、案牌、官署筹码执行御令，不推进季度。
- 世界系统：四方省势、朝堂派系、外邦诸国、密谋暗线、刑狱案卷、舆论风向、危机烈度、危机钟会随选择和时间推进变化。
- 串联玩法：密谋暴露会生成案卷；审案、宽赦、禁谣、宣判会改变舆论；舆论和未结案件会反过来触发季度奏报，影响朝稳、民心、名望和危机钟。
- 视觉资产：使用生成式高质量卡通宫廷背景、朝代画卷和角色立绘组，位于 `web/assets/`。
- Web 界面：卡通宫廷游戏界面、朝代选择卡、人物头像、属性条、危机钟、省份/派系面板、起居注和结局重开。
